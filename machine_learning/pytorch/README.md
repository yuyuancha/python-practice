# PyTorch 練習

## 環境

-   Python: `3.12.9`(搭配 `Anaconda` 使用)。

## 使用套件

### PyTorch

透過 `PyTorch` 官網設定對應設備欲載入指令。

### tensorboard

用於資料繪製圖表。

```shell
pip install tensorboard
```

#### 開啟 tesnsorboard 方法

```shell
tensorboard --logdir={資料夾名稱} --port={開啟監聽的端口}
```

### opencv-python

```shell
pip install opencv-python
```
