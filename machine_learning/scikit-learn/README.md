# 練習 scikit-learn 套件: 訓練模型

## 目的

練習透過 `python` 實作 `machine learning`。

以 `scikit-learn` 套件訓練出 `y = 10x - 12` 的線性學習模型。

## 事前準備

### 安裝套件

#### scikit-learn

```shell
pip install -U scikit-learn
```

#### matplotlib

- 繪製視覺化圖表觀察數據使用。

```shell
pip install matplotlib
```

#### panda

```shell
pip install panda
```

#### seaborn

```shell
pip install seaborn
```
